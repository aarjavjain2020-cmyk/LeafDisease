How this scan works

Every pixel is converted from red-green-blue values into hue, saturation, value — the same color space OpenCV's cv2.cvtColor uses. Pixels in the healthy green hue band count as leaf tissue; pixels in the yellow-brown band, or very dark low-value pixels, count as lesion tissue. Severity is lesion pixels divided by total leaf pixels. No model, no internet connection, no upload required — everything happens on this device.
